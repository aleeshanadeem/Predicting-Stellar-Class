# 🌌 Predicting Stellar Class | Kaggle Playground S6E6

**Competition:** [Playground Series S6E6]([https://www.kaggle.com/competitions/playground-series-s6e6](https://www.kaggle.com/code/nalisha/predicting-stellar-class-multi-model-ensemble))

### Overview
Multi-class classification problem to predict whether a celestial object is a **Galaxy**, **Star**, or **Quasar (QSO)** using SDSS data.

### Key Features Used
- Redshift (most important)
- Color indices (`u-g`, `g-r`, `r-i`, `i-z`)
- Magnitude features
- Spectral Type & Galaxy Population

### Models Used
- XGBoost
- LightGBM
- Random Forest
- Voting Ensemble
